# SpaceRacing

## ゲーム概要
疾走感と宇宙らしさにフォーカスした、
ロケットで宇宙空間を駆け抜けるレースゲームです。

プレイヤーは機体を操作し、
コース上を高速で移動しながらゴールを目指します。

## 制作時に目指したこと・意識したこと

### ・進行度を正確かつ自然に伝えるUI設計
レースゲームにおいて重要となる進行度表示について、
プレイヤーに違和感を与えないような計算と表示を意識しました。

### ・操作と一体化したAudio設計
入力に応じて音が変化することで、
プレイヤーにスピード感と操作感を直感的に伝える設計を目指しました。

### ・チーム開発における可読性の確保
自分の担当外のコードも含め、
他メンバーが追いやすい構造になるよう整理・コメント付けを行い、
チーム全体の開発効率向上を意識しました。

---

## そのために行ったこと

### ① 進行度UIの計算処理
Spline上の距離を基準に進行度を算出する仕組みを実装しました。  

- 距離ベースでの進行度算出  
- 値の急激な変化（値飛び）を抑える補間処理  
- 逆走時の不自然な表示を防ぐ最大到達距離管理  

これにより、常に安定した進行度表示を実現しています。

---

### ② Audio制御の実装
プレイヤー入力に応じてエンジン音を変化させるシステムを構築しました。  

- 低域・高域のエンジン音を分離  
- 回転音を別トラックとして管理  
- 入力値を平滑化し、ピッチと音量に反映  

これにより、操作に連動したリアルなサウンド表現を実現しています。

---

### ③ BGM管理システムの設計
BGMの再生・停止・切り替えを一つのActorに集約しました。  

- レベルに配置して使用する設計  
- ゲーム開始時にデフォルトBGMを自動再生  
- 状況に応じたBGM切り替え対応  

シンプルかつ拡張しやすい構造を意識しています。

---

### ④ コード整理・可読性向上
既存のプレイヤー制御クラスに対して、

- 関数の並び整理  
- コメントの追加  
- 処理単位の明確化  

を行い、チームメンバーが理解しやすい構造に改善しました。

---

## 操作方法（コントローラー）
本作はゲームコントローラーでの操作を想定しています。（※Xboxコントローラー準拠）

- 移動（機体の向き）：左スティック  
- カメラ操作　　　　：右スティック  
- 機体回転　　　　　：LB / RB  
- 前進　　　　　　　：RT  
- 減速　　　　　　　：LT  
- 決定　　　　　　　：Bボタン
