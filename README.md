# SpaceRacing

## ゲーム概要
疾走感と宇宙らしさにフォーカスした、
ロケットで宇宙空間を駆け抜けるレースゲームです。

プレイヤーは機体を操作し、コース上を高速で移動しながらゴールを目指します。

## 制作時に目指したこと・意識したこと

### ・違和感のない進行度表示
レースゲームにおいて重要な進行度表示について、
プレイヤーに違和感を与えない自然な挙動を目指しました。

### ・操作と一体化したAudio表現
入力に応じて音が変化することで、
プレイヤーがスピード感や操作感を直感的に感じられる設計を意識しました。

### ・チーム開発における設計と進行管理
実装だけでなく、仕様書作成やスケジュール管理にも取り組み、
チーム全体が効率よく開発できる体制を意識しました。

---

## そのために行ったこと

### ① 進行度UIの実装
Spline上の最近点から距離を取得し、進行度を算出する仕組みを実装しました。

- 距離ベースでの進行度算出  
- 別区間への吸い込みによる値飛びを抑える補間処理  
- 進行方向と速度の内積による逆走判定  
- 最大到達距離を保持し、逆走時でも進行度が戻らない設計  

これにより、常に安定した進行度表示を実現しています。

---

### ② Audio制御の実装
プレイヤー入力に応じてエンジン音が変化するシステムを構築しました。

- 推力入力・回転入力を正規化して取得  
- `FInterpTo` による入力値の平滑化  
- 低域・高域のエンジン音を分離  
- ピッチ・音量をリアルタイムに反映  

これにより、操作と連動した自然なサウンド表現を実現しています。

---

### ③ BGM管理システムの設計
BGMの再生・停止・切り替えを専用のActorに集約しました。

- `UAudioComponent` を用いた一元管理  
- Tickを無効化し、イベント駆動で制御  
- フェードイン・フェードアウト対応  
- レベル配置による管理設計  

これにより、シンプルで拡張しやすいBGM管理を実現しています。

## 操作方法（コントローラー）
本作はゲームコントローラーでの操作を想定しています。（※Xboxコントローラー準拠）

- 機体操作　：左スティック  
- カメラ操作：右スティック  
- 機体回転　：LB / RB  
- 前進　　　：RT  
- 減速　　　：LT  
- 決定　　　：Bボタン
